# RSVP Lounge — Client Intake Questionnaire

Kopyahin ang mga tanong na ito sa **Google Forms** (isang form kada okasyon, o isang form na may sections). Ang mga sagot ng kliyente dito ang ilalagay sa kanilang RSVP website — kaya kumpleto at organized agad, walang paulit-ulit na tanungan.

**Tips:**
- Gawing **Required** ang mga importanteng field (`*`).
- Para sa **photos**: hilingin ang isang **Google Drive / shared folder link** (huwag attachments — mabigat). Lagyan ng note: "Set sharing to *Anyone with the link*."
- Para sa **colors**: pwedeng pangalan lang (hal. "sage green & blush") o hex code kung meron.
- Field type nakasulat sa `[...]` — short answer, paragraph, multiple choice, atbp.

### Paalala para sa'yo (RSVP Lounge)
- **Standard** = template-based; mabilis — palit lang ng content/colors/photos.
- **Premium** = custom design + lahat ng special sections + music + larger gallery.
- Ang mga sagot dito ay direktang papunta sa data ng site (tingnan ang `src/data/demoInvitations.js` para sa istraktura).
