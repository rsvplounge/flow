# Background music para sa demos

Ilagay dito ang MP3 files na tutugtog sa bawat demo (i-tap ang 🔊 button sa kanto):

- `wedding.mp3` — para sa /demo/wedding
- `debut.mp3` — para sa /demo/debut
- `birthday.mp3` — para sa /demo/birthday

## Tips
- Gumamit ng **instrumental** at **royalty-free** na kanta (YouTube Audio Library, Bensound).
- Panatilihing maliit ang file (~1–3 MB). Pwede mong i-trim/compress.
- Kung iisang kanta lang muna ang gusto mo, kopyahin lang ito at i-rename sa 3 pangalan sa itaas.
- Naka-**off** ang music by default — kailangan i-tap ng bisita ang button bago tumugtog.
